
# Day 02 — Services & Tags

## 🔹 Ansible — Install & Enable Nginx

---
- name: Install and start Nginx on HireReady
  hosts: all
  become: yes

  tasks:
    - name: Install nginx
      yum:        
		name: nginx
        state: present

    - name: Enable nginx
      service:
        name: nginx
        state: started
        enabled: yes
